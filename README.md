
# 📧 Mail_Sec — анализатор фишинговых писем

**Mail_Sec** - это консольное приложение для анализа электронной почты с целью выявления фишинга, вредоносных вложений и подозрительных ссылок.  
Проект предназначен для учебных и исследовательских целей и демонстрирует комплексный подход к анализу писем.

---

##  Возможности

-  Подключение к **Gmail / Yandex Mail** через IMAP  
-  Анализ:
  - текста письма (локальная нейросетевая модель)
  - ссылок
  - вложений
  - вложенных ссылок Яндекс.Диска
-  Система оценки риска
-  Разделение писем на:
  - безопасные
  - подозрительные
  - фишинговые
  - коммерческие
-  Автоматическая загрузка вложений
-  Проверка файлов через **VirusTotal** или **Dr.Web** (На выбор)
-  Удобный интерфейс, реализованный на базе фреймворка PySide6
-  Цветовая индикация уровня угроз
-  Возможность скачивания вложений
